10x19 사이트 레이아웃 클론코딩


노마드의 css 플렉스박스 그리드 수업을 6만원 주고 결제하였기에 돈이 아까워 전부다 클론해서 올릴 예정이다. 
노마드가 알려주는 플렉스와 그리드의 개념을 제외하곤 코딩에 별다른 도움이 되질 않아 반응형까지 전부 직접 만들어야 했다.
이것만 하고나면 레이아웃 잡는 실력이 늘거라 생각한다.


1. 폰트 사이즈
  - 기본 : 16px
  - min-width 450px : 21px
  - min-width 800px : 28px
  - min-width 1000px : 35px

2. 레이아웃
  - grid, flex 
  - 최대 width : 100% (브라우저 사이즈 최대치)
  - 최대 height: 100% (브라우저 사이즈 최대치)
  - height 은 px로
  - width 은 퍼센티지로

3. 반응형 레이아웃 (미디어 쿼리)
  - min-width : 1000px
  - min-width : 926px / max-width : 999px
  - min-width : 800px / max-width : 925px
  - max-width : 799px
